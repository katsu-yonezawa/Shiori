# SHI-007 SQLite FTS5 による全文検索を実装する

## 概要

| 項目 | 内容 |
| --- | --- |
| 優先度 | P1 |
| マイルストーン | M2: 検索と整理 |
| 依存 | SHI-003, SHI-004 |

## 目的

ローカルDBに保存されたノートを高速に検索できるようにします。

## 作業内容

- FTS5 用の検索テーブルを作成する
- ノート作成、更新、削除と検索インデックスの同期方法を決める
- タイトルと本文を対象に検索する
- 検索結果の並び順を設計する

## 受け入れ条件

- タイトルと本文のキーワード検索ができる
- 削除済みノートが通常検索結果に出ない
- ノート更新後、検索結果に反映される
