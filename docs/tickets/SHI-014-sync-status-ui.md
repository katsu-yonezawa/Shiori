# SHI-014 同期状態とエラーをUIで確認できるようにする

## 概要

| 項目 | 内容 |
| --- | --- |
| 優先度 | P1 |
| マイルストーン | M4: 安全性向上 |
| 依存 | SHI-012 |

## 目的

利用者が保存・同期状態を把握できるようにし、不安なく使える状態にします。

## 作業内容

- 未同期、同期中、同期済み、失敗の表示を実装する
- 手動再同期の導線を用意する
- 最終同期時刻を表示する
- エラー表示の文言を整理する

## 受け入れ条件

- 現在の同期状態が画面上で分かる
- 同期失敗後に手動で再試行できる
- エラーが利用者にとって理解可能な文言で表示される
