# SHI-016 Web 版の成立性を確認する

## 概要

| 項目 | 内容 |
| --- | --- |
| 優先度 | P2 |
| マイルストーン | M5: 将来展開の確認 |
| 依存 | SHI-001, SHI-012 |

## 目的

将来のWeb展開に向け、共通化できる範囲と課題を把握します。

## 作業内容

- `packages/core`、`packages/ui`、`packages/editor` の再利用可能性を確認する
- SQLite 前提の処理がWebでどの程度課題になるか整理する
- Web版での保存、検索、同期の候補方式を比較する
- 必要なら小さなプロトタイプを作る

## 受け入れ条件

- Web版で再利用できる処理と作り直す処理が整理されている
- 技術的な主要リスクが明文化されている
- 次の実装判断に使える比較資料がある
