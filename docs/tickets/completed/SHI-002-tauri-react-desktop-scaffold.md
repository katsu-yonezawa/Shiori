# SHI-002 Tauri + React のデスクトップアプリ雛形を作成する

## 概要

| 項目 | 内容 |
| --- | --- |
| 優先度 | P0 |
| マイルストーン | M1: ローカル最小版 |
| 依存 | SHI-001 |

## 目的

macOS で起動できるデスクトップアプリの土台を作ります。

## 作業内容

- `apps/desktop` に Tauri + React + TypeScript の雛形を作成する
- アプリ起動、ウィンドウ表示、基本レイアウトを実装する
- Tauri 側とフロントエンド側の境界を整理する

## 受け入れ条件

- macOS 上でアプリを起動できる
- 画面にノート一覧と編集領域の仮レイアウトが表示される
- 開発サーバーとTauri起動手順が確認できる
