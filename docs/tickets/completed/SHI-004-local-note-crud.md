# SHI-004 ノートの作成、編集、削除をローカルDBで実装する

## 概要

| 項目 | 内容 |
| --- | --- |
| 優先度 | P0 |
| マイルストーン | M1: ローカル最小版 |
| 依存 | SHI-002, SHI-003 |

## 目的

ネットワークに依存せず、基本的なノート操作を完結できるようにします。

## 作業内容

- ノート作成処理を実装する
- タイトル、本文、本文形式、作成日時、更新日時を保存する
- ノート一覧と選択中ノートの読み込みを実装する
- 削除操作の入口を作る

## 受け入れ条件

- アプリを再起動しても作成済みノートが残る
- タイトルと本文を編集できる
- 更新日時が変更に合わせて更新される
- ネットワーク未接続でも操作できる
