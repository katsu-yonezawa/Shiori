# SHI-005 本文エディタと自動保存を実装する

## 概要

| 項目 | 内容 |
| --- | --- |
| 優先度 | P0 |
| マイルストーン | M1: ローカル最小版 |
| 依存 | SHI-004 |

## 目的

日常的に書き続けられる編集体験を整えます。

## 作業内容

- Markdown またはプレーンテキストの編集方式を決める
- 本文入力UIを実装する
- 数秒の debounce による自動保存を実装する
- ノート切り替え時とアプリ終了前の保存を考慮する

## 受け入れ条件

- 本文入力中に過剰なDB更新が発生しない
- ノート切り替え後も直前の編集内容が保持される
- 保存状態が利用者に分かる形で表示される
