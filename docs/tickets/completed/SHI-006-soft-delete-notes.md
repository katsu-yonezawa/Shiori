# SHI-006 削除済みノートを `deleted_at` で扱う

## 概要

| 項目 | 内容 |
| --- | --- |
| 優先度 | P0 |
| マイルストーン | M1: ローカル最小版 |
| 依存 | SHI-004 |

## 目的

同期時に削除情報を安全に扱えるよう、物理削除ではなく論理削除にします。

## 作業内容

- ノート削除時に `deleted_at` を設定する
- 通常のノート一覧から削除済みノートを除外する
- 同期対象として削除情報を扱えるようにする
- 必要に応じて削除済みノートの確認方法を用意する

## 受け入れ条件

- 削除後、通常の一覧には表示されない
- DB上には削除済みレコードが残る
- 削除操作も `sync_events` に記録できる
