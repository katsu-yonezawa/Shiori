# SHI-008 タグ付けとタグ検索を実装する

## 概要

| 項目 | 内容 |
| --- | --- |
| 優先度 | P1 |
| マイルストーン | M2: 検索と整理 |
| 依存 | SHI-003, SHI-004 |

## 目的

ノートをタグで整理できるようにします。

## 作業内容

- タグ作成、付与、解除を実装する
- ノート一覧でタグを表示する
- タグによる絞り込みを実装する
- タグ名の重複、空文字、前後空白の扱いを決める

## 受け入れ条件

- 1つのノートに複数タグを付けられる
- タグでノートを絞り込める
- タグ付けの変更が再起動後も保持される
