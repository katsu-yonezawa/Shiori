# SHI-010 macOS アプリに Supabase Auth ログインを接続する

## 概要

| 項目 | 内容 |
| --- | --- |
| 優先度 | P0 |
| マイルストーン | M3: 認証と同期 |
| 依存 | SHI-002, SHI-009 |

## 目的

デスクトップアプリからSupabaseへログインできるようにします。

## 作業内容

- メールログイン、OAuth、Magic Link などの初期方式を決める
- ログイン、ログアウト、セッション復元を実装する
- 認証状態に応じたUIを実装する
- セッション保存先と扱いを整理する

## 受け入れ条件

- アプリからログインとログアウトができる
- アプリ再起動後も必要に応じてセッションが復元される
- 未ログイン時は同期操作が実行されない
