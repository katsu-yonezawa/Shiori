# SHI-013 簡易的な自動同期を実装する

## 概要

| 項目 | 内容 |
| --- | --- |
| 優先度 | P1 |
| マイルストーン | M3: 認証と同期 |
| 依存 | SHI-012 |

## 目的

手動操作に頼りすぎず、低頻度で安全に同期できるようにします。

## 作業内容

- 一定間隔で同期する処理を実装する
- ノートを離れた時、アプリ終了時などの同期契機を検討する
- 同時に複数同期が走らないよう制御する
- オフライン時の扱いを整理する

## 受け入れ条件

- ログイン中かつオンライン時に自動同期が動く
- オフライン時に同期失敗を過剰に通知しない
- 同期中に再度同期が開始されない
