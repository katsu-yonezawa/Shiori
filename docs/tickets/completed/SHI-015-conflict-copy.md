# SHI-015 競合時にコピーを残す処理を実装する

## 概要

| 項目 | 内容 |
| --- | --- |
| 優先度 | P0 |
| マイルストーン | M4: 安全性向上 |
| 依存 | SHI-012 |

## 目的

複数端末で同じノートが更新された場合に、黙って上書きせず内容を保全します。

## 作業内容

- `version` と `updated_at` を使った競合検出を実装する
- 競合時に既存ノートを上書きせず、競合コピーを作成する
- 競合コピーのタイトル規則を決める
- 競合発生をUIで確認できるようにする

## 受け入れ条件

- 同じノートが複数端末で更新された場合に競合を検出できる
- 片方の内容が失われず、競合コピーとして保存される
- 競合コピーであることが一覧または詳細画面で分かる
