# SHI-018 タグ管理を強化する

## 概要

| 項目 | 内容 |
| --- | --- |
| 優先度 | P1 |
| マイルストーン | M2: 検索と整理 |
| 依存 | SHI-008, SHI-012 |

## 目的

タグの追加と絞り込みだけでなく、不要になったタグを自然に整理できるようにします。

## 作業内容

- タグ一覧またはタグ入力まわりから、不要なタグを削除できるUIを用意する
- タグ削除時に、該当タグとノートの関連付けも安全に削除する
- どのノートにも紐づかなくなったタグを自動で削除する
- ローカルDB、Web保存用 IndexedDB、Supabase 同期で未使用タグの扱いを揃える
- 選択中のタグが削除された場合の絞り込み状態を破綻しないようにする
- 削除や自動整理が同期イベントに反映されるか確認する

## 受け入れ条件

- ユーザーが不要なタグを明示的に削除できる
- ノートから最後のタグ付けを外したとき、どのノートにも使われていないタグが残らない
- 未使用タグはタグ一覧とタグ絞り込みに表示されない
- タグ削除後もノート本文、タイトル、他のタグ付けは失われない
- 同期後もローカルとSupabaseでタグ一覧の状態が一致する
