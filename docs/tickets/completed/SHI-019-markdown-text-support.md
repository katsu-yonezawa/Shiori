# SHI-019 Markdown テキストに対応する

## 概要

| 項目 | 内容 |
| --- | --- |
| 優先度 | P1 |
| マイルストーン | M2: 検索と整理 |
| 依存 | SHI-005, SHI-012 |

## 目的

ノート本文をMarkdownとして扱いやすくし、編集時と閲覧時の読みやすさを高めます。

## 作業内容

- 本文の保存形式として既存の `body_format = markdown` を前提に、編集UIと表示UIの役割を整理する
- 編集、プレビュー、必要なら分割表示を切り替えられるようにする
- 見出し、箇条書き、チェックリスト、リンク、引用、コードブロックなど主要なMarkdown記法を表示できるようにする
- Markdownレンダリング時に危険なHTMLやスクリプトが実行されないようにする
- 既存ノートの本文、自動保存、検索、同期の動作を維持する
- プレーンテキストとして扱うノートが必要かどうかを確認し、必要なら切り替え方針を決める

## 受け入れ条件

- Markdown形式の本文を編集し、再起動後も内容が保持される
- プレビューで主要なMarkdown記法が読みやすく表示される
- 編集中の自動保存がMarkdown対応後も動作する
- 本文検索と同期が既存どおり機能する
- Markdown内の危険なHTMLやスクリプトが実行されない
