# SHI-020 タグ入力の補完と表記ゆれ防止を実装する

## 概要

| 項目 | 内容 |
| --- | --- |
| 優先度 | P1 |
| マイルストーン | M2: 検索と整理 |
| 依存 | SHI-008, SHI-018 |

## 目的

タグ入力時に既存タグを選びやすくし、同じ意味のタグが表記ゆれで増えることを防ぎます。

## 作業内容

- タグ入力中に既存タグの候補を表示する
- キーボード操作で候補を選択できるようにする
- 大文字小文字、前後空白、全角半角など、表記ゆれの扱いを確認して必要な正規化を行う
- 既存タグと同じ名前のタグを重複作成しない
- タグ候補の表示が、ノート編集や自動保存の邪魔にならないようにする

## 受け入れ条件

- 既存タグを候補から選んでノートに付けられる
- タグ名の前後空白や大文字小文字差分だけで別タグが作られない
- タグ候補を使わずに新しいタグも入力できる
- タグ入力中でもノート本文の自動保存が維持される
- マウスとキーボードのどちらでも自然に操作できる
