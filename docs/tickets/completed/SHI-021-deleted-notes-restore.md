# SHI-021 削除済みノートの一覧と復元を実装する

## 概要

| 項目 | 内容 |
| --- | --- |
| 優先度 | P1 |
| マイルストーン | M4: 安全性向上 |
| 依存 | SHI-006, SHI-012 |

## 目的

誤って削除したノートを確認し、必要に応じて復元できるようにします。

## 作業内容

- 削除済みノートだけを表示する一覧を用意する
- 削除済みノートのタイトル、本文の抜粋、削除日時を確認できるようにする
- 削除済みノートを通常のノート一覧へ復元できるようにする
- 復元時にタグ、本文、同期状態が破綻しないようにする
- 完全削除を入れるかどうかは、初期実装では慎重に判断する

## 受け入れ条件

- 削除済みノートを一覧で確認できる
- 削除済みノートを復元すると通常のノート一覧に戻る
- 復元後も本文、タイトル、タグが保持される
- 復元操作が同期イベントに反映される
- 通常のノート一覧には削除済みノートが混ざらない
