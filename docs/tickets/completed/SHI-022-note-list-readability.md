# SHI-022 ノート一覧の見やすさを改善する

## 概要

| 項目 | 内容 |
| --- | --- |
| 優先度 | P1 |
| マイルストーン | M2: 検索と整理 |
| 依存 | SHI-004, SHI-007, SHI-008 |

## 目的

ノートを開く前に内容を思い出しやすくし、一覧から目的のノートを見つけやすくします。

## 作業内容

- ノート一覧に本文の短い抜粋を表示する
- 更新日時、タグ、同期状態など、必要な情報の見せ方を整理する
- 長いタイトルや長い本文でも一覧のレイアウトが崩れないようにする
- 検索結果やタグ絞り込み時にも読みやすい表示を維持する
- 情報量を増やしすぎず、日常的に見やすい密度に調整する

## 受け入れ条件

- ノート一覧でタイトル、本文抜粋、更新日時、タグを確認できる
- 長い文字列があっても一覧表示が崩れない
- 検索やタグ絞り込み後も同じ表示品質が保たれる
- 選択中のノートが分かりやすく表示される
- 一覧の情報量が過剰にならず、画面全体の操作性が維持される
