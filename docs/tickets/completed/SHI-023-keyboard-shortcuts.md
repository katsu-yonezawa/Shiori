# SHI-023 基本操作のキーボードショートカットを実装する

## 概要

| 項目 | 内容 |
| --- | --- |
| 優先度 | P2 |
| マイルストーン | M2: 検索と整理 |
| 依存 | SHI-004, SHI-005, SHI-007 |

## 目的

よく使う操作をキーボードだけで行えるようにし、日常的な記録や検索を少ない手数で進められるようにします。

## 作業内容

- 新規ノート作成、検索欄への移動、保存、削除などの基本ショートカットを定義する
- テキスト入力中の標準操作と衝突しないキー割り当てにする
- macOS の慣習に沿ったショートカットを優先する
- ショートカットの動作範囲を整理し、意図しない削除や保存が起きないようにする
- 必要なら設定画面ではなく、最小限のヘルプ表示で確認できるようにする

## 受け入れ条件

- キーボードで新規ノート作成と検索欄への移動ができる
- 保存や削除の操作が既存のUI操作と同じ結果になる
- 本文編集中に一般的な入力操作を妨げない
- 削除など取り消しにくい操作は誤操作しにくい設計になっている
- 主要なショートカットをアプリ内で確認できる
